# Flutter UI Internship — Starter Pack

Welcome. This folder contains everything you need to complete the screening task.

---

## Read these files in order

1. **TASK.md** — what to build, deadline, expected output
2. **design_tokens.md** — colors, typography, spacing
3. **allowed_packages.md** — which pub packages you may use
4. **starter_template.dart** — bare scaffold showing the file pattern
5. **SUBMISSION_CHECKLIST.md** — tick this off before submitting

---

## What's in this folder


---

## The big picture

- Pick **one** of the three reference screens.
- Build it in Flutter using the matching JSON file.
- The closer to the reference image, the better — but feel free to add your own creative touches.
- Submit by the deadline. Selection is based on the submitted task.

---

## Important rules

- **Do not contact us asking for clarifications about the reference images.** Make your own design decisions where the image is unclear. We want to see your judgement.
- **You may use AI tools** (Antigravity, Codex, Gemini, ChatGPT) however you like.
- **Submit even if incomplete.** A partial, well-built submission beats no submission.
- **Use only the packages listed in `allowed_packages.md`** — anything else, ask first.

---

Good luck. We're looking forward to seeing what you build.
